# 开发笔记

计划重写项目主要内容，第一步是网关、用户服务和配套的`web`，先把架子搭起来
- 使用`apifox`模拟前端请求，接口文件保存在`server/internal/http/api/apifox.json`
- 暂不引入服务注册中心
- `web`采用响应式布局，通过媒体查询实现一套代码同时适配网页和手机

第二步，计划添加一个业务服务，给项目添加一定的实用性
- 事实证明云文件并不实用，我想搞小纸条功能，就是写一张一张的小纸条，然后谁都能看，可以随机抓取，也可以查询列举
- review代码，完善脚本工具，准备好添加其他微服务功能

第三步，开始添加微服务功能，包括但不限于：
- 服务注册中心
- 配置中心
- 调用链追踪
- 限流、融断、降级
- ......

第四步，持续学习，阅读其他相关代码库，学习他们的优势部分，编写阅读笔记，并在这里手搓一遍
